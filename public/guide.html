Οδηγίες Χρήσης

Πώς να αξιοποιήσετε στο έπακρο την πλατφόρμα

Βήμα 1: Επιλογή Επιστημονικού Πεδίου

Από την αρχική οθόνη (Dashboard), επιλέξτε το Επιστημονικό Πεδίο που σας ενδιαφέρει (π.χ. 1ο Πεδίο Ανθρωπιστικών Σπουδών). Η πλατφόρμα θα σας μεταφέρει στον αντίστοιχο, προσαρμοσμένο υπολογιστή μορίων.

Βήμα 2: Εισαγωγή Βαθμολογιών (Άμεσο Feedback)

Χρησιμοποιήστε τα ειδικά πεδία με τα κουμπιά (+) και (-) για να εισάγετε τις βαθμολογίες σας στα τέσσερα πανελλαδικώς εξεταζόμενα μαθήματα. Καθώς πληκτρολογείτε, η πλατφόρμα θα υπολογίζει αυτόματα τον Z-Score σας, δείχνοντάς σας πού βρίσκεστε σε σχέση με τον μέσο όρο (μ) της χρονιάς για το εκάστοτε μάθημα!

Βήμα 3: Ρύθμιση Μακροοικονομικών Μοντέλων

Η πλατφόρμα μας επιτρέπει "what-if" αναλύσεις. Αν πιστεύετε ότι τα θέματα φέτος ήταν γενικά δυσκολότερα, μπορείτε να εισάγετε μια αρνητική μεταβολή στο πεδίο Μεταβολή Εθνικών Επιδόσεων (π.χ. -2%). Αντίστοιχα, αν ακούσατε για αύξηση των εισακτέων, προσθέστε ένα θετικό ποσοστό στο Μεταβολή Θέσεων Εισακτέων. Ο αλγόριθμος θα προσαρμόσει την εκτιμώμενη βάση αναλόγως.

Βήμα 4: Ανάλυση Σχολών

Ο πίνακας στο κάτω μέρος ανανεώνεται άμεσα δείχνοντας τα μόρια σας για τη συγκεκριμένη σχολή, τη βάση του προηγούμενου έτους, την απόκλιση, και την Πρόβλεψη Πιθανότητας Εισαγωγής (%). Μπορείτε να χρησιμοποιήσετε την αναζήτηση για να φιλτράρετε πόλεις (π.χ. "Αθήνα") ή τμήματα.

Βήμα 5: Επεξήγηση Πρόβλεψης (XAI)

Για κάθε σχολή που σας ενδιαφέρει, κάντε κλικ στο κουμπί "Επεξήγηση Πρόβλεψης". Θα μεταφερθείτε σε ένα αναλυτικό Dashboard (Explainable AI) το οποίο σας εξηγεί με διαγράμματα, κείμενο φυσικής γλώσσας, και αναλύσεις περιθωρίων ασφαλείας, πώς ακριβώς προέκυψε το συγκεκριμένο ποσοστό επιτυχίας!
